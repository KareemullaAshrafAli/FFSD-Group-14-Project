---- How To Run the Code  ----


1. Open Vscode and create a folder. Then open a terminal and run the command 'npm init'. Press enter for all the details asked and finally enter 'yes'. 

2. Then run the command 'npm install ejs nodemon express multer body-parser path cors fast-csv fs mongoose'

3. Then, Copy all the folders from the zip file to your current directory and run the command 'nodemon app.js'

4. Finally, open a web browser and enter 'localhost:2000' and press enter.

5. Now you will be successfully able to operate the project.
